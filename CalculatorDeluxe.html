Q-Learning and
SARSA Algorithms

I started this project for hands on experience using Android Studio after learning the basics of the program in a short course. The skills I developed in coding this basic calculator mostly surrounded the formatting of the calculator, though I did need to use a clever library as a mathematical engine for the calculator.

In formatting the layout of the screen, I decided it made the most sense to use a linear layout, since I would want all buttons to be equidistant and the same size (except for the "=" button). I learned how to nest linear layouts within each other, as well as the function of the layout_weight attribute, which I used to make the sizes of buttons and the layouts themselves proportional instead of static.

For the MainActivity.java file, I concatenated the corresponding number or symbol of each button clicked to a String. Then, using the Rhino library and the ScriptEngine class, the String was input and the answer was returned and displayed on the Answer TextView. For the parentheses, I used a simple toggle boolean to determine if the parenthese was an open or a close (this works for simple statements in parentheses, but fails in the case of nested parenthetical statements - will update in the future).
