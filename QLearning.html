Q-Learning and
SARSA Algorithms

Frozen Lake is a game where the player starts on one square of an X by X grid and must reach the target square. However, there are spaces on the grid where the player will lose if they land on them (holes in the ice). This was a game designed to help programmers learn basic reinforcements learning algorithms.

The point of these two algorithms is to make the program explore the game board and determine the value of each state. The program will make the smartest move based on the reward value of the next state, however there is some randomness involved, where the program may make a random move with probability epsilon, leading to helpful exploration. Through recursion, the epsilon value progressively shrinks, leading the program to use the most efficient path it has found from start to finish.
