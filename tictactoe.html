Tic-Tac-Toe
(Python)

This simple game algorithm allows the user to play against a computer opponent which makes random moves. I had fun implementing this algorithm, and if I wished to make it more in depth, I could try to make the computer opponent smarter. Perhaps I could train the computer to make certain moves based on a trained model using a machine learning algorithm, though I wonder if simply hard coding the computer's decisions would be easier, since the game board is only 3x3.
